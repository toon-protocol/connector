### Integration Test — Architecture and Approach

**Test location:** `packages/connector/test/integration/per-hop-notification.test.ts`

**Test approach:** In-process 3-connector chain using `ConnectorNode` instances with in-process `localDeliveryHandler` functions (no Docker, no HTTP BLS server needed). This follows the pattern from `multi-node-forwarding.test.ts` but adds BLS mock handlers.

**3-Connector Chain Setup:**

```
A (sender) → B (intermediate, perHopNotification: true) → C (final hop, BLS accepts)
```

- **Connector A:** Routes `g.connectorC.*` → Connector B
- **Connector B:** Routes `g.connectorC.*` → Connector C. Has `perHopNotification: true` and a mock in-process `localDeliveryHandler` that records received notifications
- **Connector C:** Is the final hop. Has an in-process `localDeliveryHandler` that returns `{ accept: true }` (blocking delivery)

**BLS mock pattern using `setLocalDeliveryHandler()`:**

```typescript
// Connector B: transit notification mock
const transitNotifications: LocalDeliveryRequest[] = [];
connectorB.setLocalDeliveryHandler(async (request: LocalDeliveryRequest) => {
  transitNotifications.push(request);
  return { accept: true }; // Response ignored for transit, but must be valid
});

// Connector C: final-hop BLS
connectorC.setLocalDeliveryHandler(async (request: LocalDeliveryRequest) => {
  return { accept: true }; // This drives the FULFILL response
});
```

**Guard clause:** Use `describe.skip` if `E2E_TESTS !== 'true'`, or run unconditionally if using in-process connectors (no Docker dependency). The existing `multi-node-forwarding.test.ts` uses `describeIfE2E` — follow the same pattern.

**Latency measurement (AC 2, sub-criterion):**

Use multiple iterations and compare averages to avoid CI flakiness from timing jitter:

```typescript
const ITERATIONS = 5;

// Baseline: perHopNotification disabled
const baselineTimes: number[] = [];
for (let i = 0; i < ITERATIONS; i++) {
  const start = performance.now();
  await sendPacketThroughA(packet);
  baselineTimes.push(performance.now() - start);
}
const baselineAvg = baselineTimes.reduce((a, b) => a + b, 0) / ITERATIONS;

// With per-hop notification enabled
const perHopTimes: number[] = [];
for (let i = 0; i < ITERATIONS; i++) {
  const start = performance.now();
  await sendPacketThroughA(packet);
  perHopTimes.push(performance.now() - start);
}
const perHopAvg = perHopTimes.reduce((a, b) => a + b, 0) / ITERATIONS;

// Fire-and-forget overhead should be negligible on average
expect(perHopAvg - baselineAvg).toBeLessThan(10); // <10ms average overhead (CI-safe)
```

  ```typescript
  const perHopEnabled = this.localDeliveryClient?.isPerHopNotificationEnabled() ?? false;
  if (perHopEnabled) {
    let dispatched = false;
    if (this.localDeliveryHandler) {
      // In-process handler path (takes priority over HTTP)
      dispatched = true;
      const transitRequest: LocalDeliveryRequest = {
        /* ... existing code ... */
      };
      this.localDeliveryHandler(transitRequest, sourcePeerId).catch((err: unknown) => {
        /* ... */
      });
    } else if (this.isLocalDeliveryEnabled() && this.localDeliveryClient) {
      // HTTP client path
      dispatched = true;
      this.localDeliveryClient
        .deliver(packet, sourcePeerId, { isTransit: true })
        .catch((err: unknown) => {
          /* ... */
        });
    }

    // Emit PER_HOP_NOTIFICATION telemetry only when a notification was actually dispatched
    if (dispatched) {
      try {
        const perHopEvent = {
          type: 'PER_HOP_NOTIFICATION' as const,
          nodeId: this.nodeId,
          destination: packet.destination,
          amount: packet.amount.toString(),
          nextHop,
          sourcePeer: sourcePeerId,
          correlationId,
          timestamp: Date.now(),
        };
        if (this.telemetryEmitter) {
          this.telemetryEmitter.emit(perHopEvent);
        } else if (this.eventStore) {
          this.eventStore.storeEvent(perHopEvent).catch((err) => {
            this.logger.warn(
              { error: err.message, correlationId },
              'Failed to store PER_HOP_NOTIFICATION event'
            );
          });
          this.eventBroadcaster?.broadcast(perHopEvent);
        }
      } catch {
        // Telemetry emission is non-blocking — swallow errors
      }
    }
  }
  ```
